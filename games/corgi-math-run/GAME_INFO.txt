Game: Corgi Math Run
Audience: Kids 3-6

Purpose:
- Train basic arithmetic (simple addition/subtraction).
- Build calm decision-making and focus.
- Improve confidence through clear visual feedback.

Core loop:
1) Player opens game menu and selects difficulty.
2) Press Start to begin run.
3) Corgi approaches obstacle equation with multiple answer options.
4) Correct answer advances progress and score.
5) Wrong answer disables that selected option and consumes attempt.
6) 3 wrong attempts total => game over.

Difficulty levels:
- easy: 10 equations
- middle: 15 equations
- complex: 20 equations

Equation rules:
- Equation keys are tracked to avoid repeats during a run.
- Options are unique (no repeated suggestions in one question).
- Option count grows near finish:
  - early: 3 options
  - middle: 4 options
  - near finish: 5 options

State and scoring:
- `score` increments by 1 per correct answer.
- `wrongAttempts` max is 3.
- `highScore` persisted in localStorage key: `corgiMathRunHighScoreV1`.
